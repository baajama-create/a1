# Barbecue Нальчик

Сайт ресторана «Барбекю» в Нальчике — мясо, гриль, роллы. Меню, корзина, заказ в WhatsApp.

## Публикация на GitHub Pages

1. Создай новый репозиторий на GitHub (например, `barbecue-nalchik`).
2. Загрузи в него файл **`index.html`** из этой папки.
3. Перейди в **Settings → Pages**.
4. В разделе **Source** выбери ветку **`main`** и папку **`/ (root)`**, нажми **Save**.
5. Через 1–2 минуты сайт будет доступен по адресу:
   `https://ТВОЙ-НИК.github.io/barbecue-nalchik`

## Что внутри

- Один HTML-файл (~2 МБ) со всеми шрифтами, фото и логикой
- Работает без интернета (фото блюд встроены)
- Корзина и данные клиента хранятся в браузере
- Заказы отправляются в WhatsApp на номер +7 (928) 710-40-60
